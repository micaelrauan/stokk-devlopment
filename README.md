# Stokk - Sistema de Gerenciamento de Estoque

## Sobre

Sistema completo de gerenciamento de estoque com controle de produtos, vendas, etiquetas e painel administrativo.

## Tecnologias

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
- Supabase

## Como rodar

```sh
# Clonar o repositório
git clone https://github.com/micaelrauan/stokk-development.git

# Entrar no diretório
cd stokk-development

# Instalar dependências
bun install

# Copiar e configurar variáveis de ambiente
cp .env.example .env.local
# Edite .env.local com suas credenciais do Supabase

# Iniciar servidor de desenvolvimento
bun dev
```
